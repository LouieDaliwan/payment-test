## Installing the project

1. Server requirements
    - PHP 8.1 or higher
    - Node 20.3.0
2. Clone the repository
3. Install the dependencies
    - `composer install`
    - `npm install`
4. Create a `.env` file
    - `cp .env.example .env`

## Running the test
   -- /vendor/bin/phpunit

Please to see the controllers and routes, check the `routes/api.php` file.
